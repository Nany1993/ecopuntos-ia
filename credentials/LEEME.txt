# SmartSort — persistencia local

Todos los datos del piloto se guardan en SQLite:

  data/smartsort.db

Tablas principales:
  - colaboradores      (ID Telegram único + nombre)
  - catalogo_canecas
  - registro_intentos
  - puntos_colaborador
  - sesiones

Datos demo: python -m scripts.seed_demo_data --limpiar
